Statistical terms for web developers: Types of data

정량 데이터 (Quantitative Data)

수치로 표현할 수 있는 데이터

이산형 데이터 (Discrete Data)

연속성이 없는 데이터입니다. 예를 들어, 웹사이트의 일일 방문자 수나, 사용자가 어떤 버튼을 몇 번 클릭했는지 등이 있습니다.

연속형 데이터 (Continuous Data)

연속된 구간에서 어떤 데이터입니다. 웹개발에서는 페이지 로딩 시간, 사용자의 세션 시간 등이 이에 해당됩니다.

정성 데이터 (Qualitative Data)

수치로 표현되지 않는 데이터

범주형 데이터 (Categorical Data)

명확한 범주나 그룹에 속하는 데이터입니다. 예를 들어, 사용자의 성별, 혈액형, 국적 등이 있습니다.

순서형 데이터 (Ordinal Data)

범주 간에 순서가 있는 데이터입니다. 웹개발에서는 사용자의 서비스 만족도 설문조사 결과나 제품 리뷰의 별점 등이 이에 해당됩니다.

정량 (定量): 量 숫자로 매겨질수 있는.
정성 (定性): 性 인간의 주관적인 판단.
