Maintain consistency in hashCode()

hashCode() 메서드는 객체의 해시 코드 값을 반환하는 역할을 합니다.
“이펙티브 자바” 책에서는 hashCode() 메서드에서 일관성을 유지하는 것의 중요성을 강조합니다.

hashCode() 메서드에서 일관성을 유지하면,
동일한 객체에 대해 호출될 때마다 동일한 값을 리턴하므로,
객체를 확인을 용이하게 할수 있습니다.
