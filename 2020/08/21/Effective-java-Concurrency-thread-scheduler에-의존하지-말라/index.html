스레드 스케줄러에 의존하지 마라

스레드 스케줄러는 운영체제의 일부로, 실행 중인 여러 스레드 중 어떤 스레드를 CPU에 할당할지 결정합니다.
스레드 스케줄러는 다양한 정책을 사용하며, 이 정책은 운영체제마다 다를 수 있습니다.

ExecutorService는 스레드 스케줄러에 직접 의존하지 않아도 스레드를 효과적으로 제어할 수 있게 합니다.
