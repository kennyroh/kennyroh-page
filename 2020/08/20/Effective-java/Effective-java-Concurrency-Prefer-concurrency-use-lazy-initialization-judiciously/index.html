lazy initializatio은 신중하게

lazy initialization란 객체의 초기화를 뒤로 미루는 기법을 말합니다.
동시성 환경에서는 lazy initialization를 신중하게 사용해야 합니다
ConcurrentHashMap의 computeIfAbsent 메소드를 사용하면, 지연 초기화를 안전하게 구현할 수 있습니다.

지연 초기화를 동시성 환경에서 사용할 때는 주의가 필요합니다.
동시성 유틸리티를 사용하면 문제 해결에 도움이 됩니다.
