열거 타입 활용하기

열거 타입은 한정된 값 집합을 표현하는 데 사용합니다.

열거 타입을 사용하면 코드의 가독성을 높이고, 오류를 줄일 수 있습니다.
열거 타입은 자신의 값 집합을 명확하게 제한하므로,
예상치 못한 값이 발생하는 것과 오타로 인한 에러를 방지할 수 있습니다.
