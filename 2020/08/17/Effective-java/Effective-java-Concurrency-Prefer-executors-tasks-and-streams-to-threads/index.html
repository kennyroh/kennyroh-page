Kenny Roh
2020-08-17
java

스레드보다는 executors, 태스크, 스트림을 선호하라.

자바에서는 스레드를 직접 관리하는 대신 executor를 사용하는 것이 좋습니다.
executor는 스레드의 생성, 관리, 제어를 담당하며, 작업을 병렬로 실행하거나 스케줄링하는 등의 고수준 기능을 제공합니다.

이 코드에서는 ExecutorService를 사용하여 두 개의 태스크를 병렬로 실행합니다.
마지막으로 shutdown 메서드를 호출하여 모든 태스크가 완료되면 executor 서비스를 종료합니다.
