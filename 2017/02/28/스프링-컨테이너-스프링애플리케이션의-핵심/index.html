spring framework 스프링 컨테이너 스프링애플리케이션의 핵심

Kenny Roh
2017-02-28
java

Spring container

스프링 프레임워크에는 ‘스프링 컨테이너’라는 핵심 개념이 있습니다.

스프링 컨테이너는 애플리케이션의 빈들을 관리하고, 필요한 의존성을 주입하는 역할을 합니다.
이를 통해 개발자는 복잡한 의존성 관리에서 벗어나 비즈니스 로직에 집중한다고 합니다.

저는 이게 복잡한 의존성관리에서 벗어나는지 잘 모르겠습니다.
